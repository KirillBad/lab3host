Какие можно получить доступ бесплатно?

На просторах интернета полно различных сайтов, сервисов, ботов, которые предоставляют доступ к chatGPT3.5 и chatGPT4.

Большинство из этих сервисов являются условно бесплатными. Спустя определенное количество обращений, возможность вновь отправить запрос станет недоступной и всплывет окно об оплате.

g4f

g4f - это утилита, имеющая открытый исходный код, распространяется под лицензией GNU GPL v3.

Смысл данной python библиотеки в том, что она собирается и обновляет список всех сервисов, которые предоставляют условно бесплатный доступ к chatGPT. С помощью данной библиотеки можно как вызывать API запросы к данным ресурсам, так и общаться в привычном интерфейса, ведь у данного инструмента имеется свой GUI.

Данное решение хорошо подойдет для личного использования, но не более. Ведь у большинства из таких сервисов очень жесткие ограничение по количеству обращений в минуту.
